# TravelQuest 🌍

En reseinspirations-app där du kan utforska länder och spara dina drömresor i en bucketlist.

## Tekniker

- Vite + React
- React Router
- Zustand
- RestCountries API

## Kom igång

1. Klona repot
2. Installera beroenden:
```
   npm install
```
3. Starta utvecklingsservern:
```
   npm run dev
```
4. Öppna `localhost:5173` i webbläsaren

## Funktioner

- Utforska länder från hela världen
- Spara länder till din bucketlist
- Se detaljinfo om varje land
